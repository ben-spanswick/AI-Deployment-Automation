Quick Start

ChromaDB is a vector database that stores embeddings and performs similarity searches. Here's how to use it:

Python Example:

import chromadb

# Connect to ChromaDB
client = chromadb.HttpClient(host='0.0.0.0', port=8000)

# Create a collection
collection = client.create_collection("my_collection")

# Add documents with embeddings
collection.add(
    documents=["This is a document", "This is another document"],
    metadatas=[{"source": "my_source"}, {"source": "my_source"}],
    ids=["id1", "id2"]
)

# Query the collection
results = collection.query(
    query_texts=["This is a query document"],
    n_results=2
)

REST API Example:

# Create a collection
curl -X POST http://localhost:8000/api/v1/collections \
  -H "Content-Type: application/json" \
  -d '{"name": "my_collection"}'

# Add embeddings
curl -X POST http://localhost:8000/api/v1/collections/my_collection/add \
  -H "Content-Type: application/json" \
  -d '{
    "embeddings": [[1.0, 2.0, 3.0]],
    "documents": ["Document text"],
    "ids": ["doc1"]
  }'
